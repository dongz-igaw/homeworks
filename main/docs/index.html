FOOTER는 웹 페이지 하단에 게시되는 영역입니다.
상단 페이지 이동을 돕는 플로팅 버튼과, 카피라이터, 연락 정보와 버전에 대한 내용을 기재하실 수 있습니다.
.works-footer 클래스를 이용하여 해당 영역을 구성 가능하며, 해당 영역은 구조상 .works-body 하위에 포함되어야 합니다.
